# Following

## Rust

```dataview
table
	without ID
	file.link as name,
	file.mtime as "last edit",
	topics,
	status
from [[Rust]]
where note_type != "MOC"
and file.mtime >= date(now) - dur("1 weeks")
and status != "🧑"
sort file.mtime desc
```

## VoiceTube

```dataview

	without ID
	file.link as name,
	file.mtime as "last edit",
	file.folder as folder,
	status
where note_type != "MOC"
and file.ctime >= (date(now) - dur("1 weeks"))
```
